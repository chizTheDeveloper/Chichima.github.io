Project Lumi

Project Summary

I particapted as a technical artist to make an underwater game in Unity; a single-player 3D game with three levels and a Boss Fight. The main goal is to lift a curse by building a powerful object by exploring the ocean. The game style focuses on visual elements and creates an immersive feeling of being under water. This project took 4 months to complete as we worked in a group of 6, pushing through bugs and design difficulties to accomplish our first underwater 3d game.

GAME OVERVIEW :

  • Single player 3D game
  • Three Levels + Boss Fight
  • Goal: Lift curse by building a powerful object
  • Based on exploration: Heavily relies on visual style

CHALLENGES:

  • Textures - struggled to put some in Unity
  • Shader graph textures
  • Exporting animations
  • UI design and overall styles
  • Underwater environment
  • Making dialogue efficient

Story

All humankind is placed under a curse by a dark force, forcing them to live underwater in the form of an animal spirit. The player meets an NPC that helps them go through the levels, but that NPC turns out to be the evil final boss.

Precediual generation & environmental design :

Provided procedural generation to speed up workflow by enabling random spawning of plants in the environment. The same concept was used to generate schools of fish that clustered and separated overtime. This made it easier to place assets in the environment regardless of bumps in the surface as the procedural generated game object would rotate to match the orientation of the surface.
