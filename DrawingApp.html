Drawing app

Project summary

Develp a painting application using andriod studio using API 28, this was a solo project in which was more focused on core features so I apologize for the bad UI.

Core Features

  • Multi-touch drawing:

    should support at least three simultaneous contact points to draw on the screen. 3 fingers on the screen create a triangle.

  • Shape drawing: Support any three of the following four shapes.

    • Lines: – touching a point sets the starting point for a line. Each subsequent point touched draws another line from the previous touched point to this touched point.

    • Rectangle – touching a point sets one corner of a quad in the current drawing colour. Touching a second point sets the other corner.

    • Circle – Touching one point sets the centre of a circle, touching the second point sets its radius.

  • Clear the screen:

    Add a control to clear the screen, removing all drawn elements, resetting the screen bank to a blank canvas.

  • Eraser:

    Add a control to change to eraser mode. In eraser mode, touching the screen should overwrite (e.g., in background colour) previously drawn strokes.

  • Undo last stroke:

    Add a control to undo the most recently drawn stroke. Repeatedly hitting undo should continue to undo the next most recent stroke. How you address multi-touch here is up to you (i.e., whether three simultaneously strokes remove at the same time, or each in sequence).

  • Implement a nice colour picker:

    Add a colour picker control that supports a full range of colours

  • Zoom Effect

    Zoom in/out with a pinch gesture
